# Evidence Directory

This directory contains raw evidence and artifacts collected during research.

## Subdirectories

- **screenshots/** - Screenshots of websites, social media posts, documents, etc.
- **documents/** - PDF files, text documents, official records, etc.
- **audio/** - Audio recordings, podcasts, voice samples, etc.
- **video/** - Video recordings, interviews, media clips, etc.
- **images/** - Photographs, diagrams, visual evidence, etc.
- **code/** - Source code, scripts, malware samples, etc.

## Evidence Documentation

Each evidence file should include:
- Descriptive filename with date (YYYY-MM-DD)
- Source attribution
- Timestamp of collection
- Context and relevance notes

## Naming Convention

Use the following format: `YYYY-MM-DD_[type]_[description].[ext]`

Examples:
- `2020-12-15_screenshot_naomi_seibt_tweet.png`
- `2024-06-20_document_hbo_bitcoin_documentary.pdf`
- `2022-11-30_video_deepfake_example.mp4`

## Privacy and Legal Considerations

- Ensure evidence collection complies with applicable laws
- Archive publicly available content only
- Do not include private or confidential information
- Respect copyright and intellectual property rights
