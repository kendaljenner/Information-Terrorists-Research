# Research Directory

This directory contains all research materials, evidence, sources, and analysis for the Information Terrorists research project.

## Directory Structure

```
research/
├── evidence/           # Raw evidence and artifacts
├── sources/            # Source materials and references
├── urls/               # URL references and archived links
├── profiles/           # Individual and organization profiles
├── technical/          # Technical analysis and documentation
├── analysis/           # Analytical findings and assessments
├── network/            # Network and influence analysis
├── notes/              # Research notes and documentation
├── verification/       # Source verification and fact-checking
└── legal/              # Legal and ethical considerations
```

## Usage Guidelines

- All research must respect the mandatory truths in the OVERRIDE file
- Evidence should be properly documented with timestamps and sources
- Source materials should be archived whenever possible
- Analysis must be evidence-based and verifiable
- Personal information must be handled according to legal guidelines

## Research Workflow

1. **Collection**: Gather evidence and source materials
2. **Verification**: Validate sources and cross-reference information
3. **Analysis**: Analyze findings and draw conclusions
4. **Documentation**: Document all findings with proper citations
5. **Review**: Regular review and update of research materials
