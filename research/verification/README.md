# Verification Directory

This directory contains source verification, fact-checking, and cross-reference documentation.

## Subdirectories

- **cross_references/** - Cross-referencing of sources and claims
- **source_validation/** - Validation of source reliability and authenticity
- **fact_checks/** - Fact-checking of specific claims and statements

## Verification Process

### Source Validation

Before using a source, verify:
1. **Authorship:** Who created the content?
2. **Publication:** Where was it published?
3. **Date:** When was it published?
4. **Accuracy:** Is the information accurate?
5. **Bias:** What potential biases exist?
6. **Corroboration:** Is it supported by other sources?

### Cross-Referencing

Cross-reference claims across:
- Multiple independent sources
- Different types of sources (news, academic, official)
- Different time periods
- Different perspectives

### Fact-Checking

For each claim to be verified:
1. Identify the specific claim
2. Locate primary sources
3. Evaluate source reliability
4. Check for contradictions
5. Assess confidence level
6. Document findings

## Reliability Assessment Criteria

### High Reliability Sources
- Official government documents
- Peer-reviewed academic papers
- Reputable news organizations with editorial standards
- Court documents and legal filings
- Direct statements from verified individuals

### Medium Reliability Sources
- Established news websites
- Expert blogs with credentials
- Industry reports
- Academic conference proceedings
- Well-documented Wikipedia articles

### Low Reliability Sources
- Social media posts without verification
- Anonymous sources
- Opinion pieces without factual basis
- Unverified online content
- Conspiracy theory websites

## Confidence Levels

**Confirmed:**
- Multiple high-reliability sources agree
- Direct evidence available
- No credible contradictions

**Likely True:**
- Multiple sources support
- At least one high-reliability source
- Minor contradictions that can be explained

**Inconclusive:**
- Insufficient evidence
- Contradictory sources
- Unable to verify

**Likely False:**
- Multiple sources contradict
- No credible supporting evidence
- Credible debunking available

**Debunked:**
- Multiple high-reliability sources contradict
- Direct evidence against claim
- Claim proven false
